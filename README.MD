Hello and Welcome to "Defeat The Evil Wizard"
=============================================

This is my final project for the Advanced Python Module.

In this project you will find:

-  A turn based battle game using OOP concepts that extends off of a starter code.

- Choose your class from the following: Paladin, Archer, Mage, Warrior

- Fight and defeat an Evil Wizard using turn a turn based battle system in the terminal.

- Mechanics include:
    - Special Abilities
    - Capped Healing
    - Damage dealt in a range
    - An Evil Wizard who regenerates health each turn after attacking.


===========
How to Play
===========

- Click the button on the top right of the window to run the program.

- Select a class, add a name, and start battling using the options given by selecting the corresponding number!

- Can you defeat the Evil Wizard?
